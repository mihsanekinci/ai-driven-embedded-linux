== Temel Kavramlar ==

[file_operations]
  Cekirdek ile kullanici alani arasindaki kopru.
  open/read/write/release fonksiyon isaretcilerini barindirir.
  Kullanici read(fd,...) yazdigi an cekirdek temp_fops.read'i cagirir.

[alloc_chrdev_region]
  Dinamik major numarasi atar — sabit numara secmek yerine cekirdege birakir.
  Cakisma riski yoktur. MAJOR() / MINOR() makrolariyla parcalanir.

[copy_to_user / copy_from_user]
  Cekirdek <-> kullanici bellek kopyalama. Dogrudan isaretci kullanilamaz;
  bu fonksiyonlar adres dogrulama ve hata yonetimi yapar.

[printk]
  Cekirdek log mekanizmasi. printf degil; libc kullanilmaz.
  KERN_INFO / KERN_ERR onekleriyle log seviyesi belirlenir.
  Cikti: dmesg veya /dev/kmsg

[class_create + device_create]
  /sys/class/temp/ olusturur. udev/mdev bu sinifi izler ve
  /dev/temp_sensor dosyasini otomatik yaratir.

== Akis Ozeti ==
  insmod  → module_init → alloc_chrdev_region → cdev_add
          → class_create → device_create → /dev/temp_sensor hazir

  open()  → temp_open    → printk("acildi")
  read()  → temp_read    → copy_to_user("25.3 C\n")
  write() → temp_write   → copy_from_user → printk(log)
  close() → temp_release → printk("kapatildi")

  rmmod   → module_exit → device_destroy → class_destroy
          → cdev_del → unregister_chrdev_region
